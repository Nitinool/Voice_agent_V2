# 天气查询

用户询问某个城市的天气、温度、是否下雨、穿什么等问题时，调用 `get_weather` 工具。

## 使用要点

- **单城市**：用户问"北京天气怎么样"，直接调用 `get_weather(city="北京")`，工具返回当前温度、湿度、空气质量、今日天气类型和温度范围。你基于结果用自然语言回答。
- **多城市对比**：用户问"北京和上海哪个更凉快"，**分别调用两次** `get_weather`（一次北京、一次上海），拿到两个城市的数据后**自己对比**再回答。不要只查一个城市就回答。
- **穿衣建议**：用户问"今天该穿什么"，查到温度范围后给出建议。
- **不支持的城市**：工具会返回 `supported_cities` 列表，告诉用户支持哪些城市。

## 支持的城市

北京、上海、深圳、广州、杭州、南京、成都、重庆、武汉、西安、天津、苏州、长沙、郑州、青岛、厦门、大连、昆明、哈尔滨、济南。
