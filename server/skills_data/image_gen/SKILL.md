# 图片生成

用户要求画图、生成图片、生成插画时，调用 `generate_image` 工具。

## 使用要点

- **触发场景**：用户说"画一只猫"、"生成一张风景图"、"帮我画个插画"等。
- **prompt 要详细**：把用户的描述转化成详细的画面描述传给工具，如用户说"画只猫"，你可以补成"一只在草地上奔跑的橘猫，阳光明媚，写实风格"。
- **生成后**：工具会自动把图片发给用户，你只需简短说"图生成好了，已发给你"或描述一下你画了什么，**不要重复念图片 URL**。
- **不要调用工具的场景**：用户只是聊天提到图片，没要求生成时不要调。
